getWeatherData

  • getWeatherData(zipCode: string): Promise<unknown>
    • Defined in optimization/construct-optimization-feed.ts:102

    Get the weather forcast data for the next 5 days in an intervall of 3H The data are requested from open weather map in the following structure:

    • 3H blocks. 8 times 3H Blocks = 24H day. 1. block 00:00 - 03:00, 2.block 03:00 - 06:00
    • The first block time (dt) of the request is the next 3H interval. For example: if the weather forcast is requested at 10:06, then the the timestamp of the first block is 12:00.

    Parameters

    • zipCode: string

      zip code for weather forcast
